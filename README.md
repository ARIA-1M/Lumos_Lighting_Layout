<h1 align="center">Lumos Lighting Layout</h1>

<h2>Описание</h2>

<p>
  Проект "Lumos" представляет собой макет интеллектуальной системы управления освещением, разработанный на платформе Arduino.
</p>

<p>
  Устройство реализует концепцию «Умный дом» и позволяет управлять светом несколькими способами: вручную, по расписанию, а также автоматически — в зависимости от уровня освещенности, звука (хлопок) и движения в помещении.
</p>

<p>
  Макет оснащен цветным сенсорным TFT-экраном для удобного выбора режимов и настройки параметров, а в качестве источника света используется RGB-светодиод с регулировкой яркости и цветовой температуры.
</p>

<h2>Функциональная схема</h2>

<p>
  Основой макета является микроконтроллер Arduino Mega, к которому подключены все периферийные модули. Управление работой системы осуществляется через сенсорный TFT-дисплей, который служит интерфейсом пользователя.
</p>

<p>
  Микроконтроллер обрабатывает сигналы от подключенных датчиков и, в зависимости от выбранного режима, управляет свечением RGB-светодиода.
</p>

<h3>Программные блоки</h3>

<ul>
  <li>
    <b>Главное меню:</b> Ручное включение и выключение света.
  </li>
  <li>
    <b>Настройка света:</b> Выбор цвета, регулировка яркости и теплоты белого цвета.
  </li>
  <li>
    <b>Планинг дня:</b> Автоматическое изменение цвета и яркости освещения по заданному расписанию (в демонстрационном режиме время ускорено: 1 час = 1 секунде).
  </li>
  <li>
    <b>Управление по звуку:</b> Включение и выключение света по хлопку.
  </li>
  <li>
    <b>Управление по движению:</b> Автоматическое включение света на заданное время при обнаружении движения.
  </li>
  <li>
    <b>Управление по освещенности:</b> Автоматическое включение света при низком уровне естественного освещения и выключение при высоком.
  </li>
</ul>

<h2>Принципиальная схема</h2>

<p>
  Проект построен на базе микроконтроллера <b>Arduino Mega 2560</b>. Все компоненты подключаются непосредственно к его выводам. Принципиальная схема определяет соединения между контроллером и следующими модулями:
</p>

<ul>
  <li>
    <b>RGB-светодиод</b> (с общим катодом) — подключен к цифровым выводам с ШИМ для регулировки яркости каждого цвета.
  </li>
  <li>
    <b>TFT дисплей 2.4"</b> — подключается к соответствующим цифровым выводам для организации SPI-интерфейса и управления сенсорным экраном.
  </li>
  <li>
    <b>Датчик движения</b> (инфракрасный) — подключается к цифровому входу и питанию 5V.
  </li>
  <li>
    <b>Датчик звука</b> — подключается к аналоговому входу (для измерения уровня шума) и к цифровому входу (для сигнала о превышении порога).
  </li>
  <li>
    <b>Датчик освещенности</b> — подключается к аналоговому входу.
  </li>
</ul>

<h2>Описание подключений</h2>

<h3>RGB светодиод</h3>

<ul>
  <li>
    <b>Красный канал (R):</b> Пин 44 (с ШИМ)
  </li>
  <li>
    <b>Зеленый канал (G):</b> Пин 45 (с ШИМ)
  </li>
  <li>
    <b>Синий канал (B):</b> Пин 46 (с ШИМ)
  </li>
  <li>
    <b>Общий катод:</b> GND
  </li>
</ul>

<h3>Датчик движения (HC-SR501)</h3>

<ul>
  <li>
    <b>VCC:</b> 5V
  </li>
  <li>
    <b>GND:</b> GND
  </li>
  <li>
    <b>OUT:</b> Пин 48
  </li>
</ul>

<h3>Датчик звука</h3>

<ul>
  <li>
    <b>VCC:</b> 5V
  </li>
  <li>
    <b>GND:</b> GND
  </li>
  <li>
    <b>Аналоговый выход (A0 на модуле):</b> Аналоговый пин A15 (14)
  </li>
  <li>
    <b>Цифровой выход (D0 на модуле):</b> Пин 47
  </li>
</ul>

<h3>Датчик освещенности</h3>

<ul>
  <li>
    <b>VCC:</b> 5V
  </li>
  <li>
    <b>GND:</b> GND
  </li>
  <li>
    <b>Аналоговый выход (S):</b> Аналоговый пин A14 (13)
  </li>
</ul>

<h3>TFT дисплей 2.4" (с сенсорным экраном)</h3>

<p>
  Дисплей устанавливается на штатные разъемы платы Arduino Mega. Адресные пины для работы с экраном (CS, CD, WR, RD, RESET) и тачскрином (XP, YP, XM, YM) настраиваются программно в соответствии с используемой библиотекой.
</p>

<h2>Особенности реализации</h2>

<ul>
  <li>
    <b>Язык программирования:</b> C++ (среда разработки Arduino IDE)
  </li>
  <li>
    <b>Используемые библиотеки:</b> SPFD5408_Adafruit_GFX, SPFD5408_Adafruit_TFTLCD, SPFD5408_TouchScreen, GRGB
  </li>
</ul>
